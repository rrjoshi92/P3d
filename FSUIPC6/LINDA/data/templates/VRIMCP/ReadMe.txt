Templates Files

These files provides templates for the Configuration files saved in /linda-cfg/config-mcpX.lua files for each VRInsight Combo panel. These cover the EFIS, MCP, RADIO and USER sub-panels. The suffix number refers to the panel types:

1 = MCP1(original), 2 = MCP2(Boeing) and 3= MCP2a(Airbus)

DO NOT EDIT OR CHANGE THESE FILES
